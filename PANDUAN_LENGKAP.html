🎯 Penjelasan Fitur Detail

1. Home Page (home.php)

Halaman public yang bisa diakses oleh semua orang tanpa login.

Fitur:

  • Header dengan logo, search bar, dan navigasi
  • Navbar untuk filter kategori produk
  • Banner promosi dengan gradient
  • Grid produk dinamis 4 kolom (responsive)
  • Search realtime dengan JavaScript
  • Tambah keranjang (perlu login)
  • Footer informatif

Desain: Terinspirasi dari Walmart.com dengan warna biru #0071ce dan gold #ffc220

2. Dashboard Admin (dashboard_admin.php)

Dashboard khusus untuk admin dengan statistik dan manajemen lengkap.

Fitur:

  • Sidebar navigasi tetap dengan menu lengkap
  • Header dengan tanggal dan avatar user
  • Stats cards: Total Pesanan, Pengguna, Produk, Revenue
  • Tabel pesanan terbaru dengan status badge
  • Modal edit profil dengan upload foto
  • All-in-one sidebar navigation

Akses menu: Dashboard, Tambah Produk, Pesanan, Chat, Edit Profil, Logout

3. Dashboard User (dashboard_user.php)

Dashboard personal untuk pengguna reguler dengan riwayat pesanan.

Fitur:

  • Sidebar navigasi untuk user
  • Stats: Total Pesanan, Pengeluaran, Keranjang
  • Tabel pesanan personal
  • Section informasi profil dengan avatar
  • Modal edit profil dengan upload foto
  • Link quick access ke berbagai halaman

Akses menu: Dashboard, Keranjang, Pesanan, Belanja, Chat, Edit Profil, Logout

4. Edit Profil & Upload Foto (update_profile.php)

API endpoint untuk menangani update profil dan upload foto profil.

Field yang bisa diupdate:

  • Email (varchar 100)
  • Nomor Telepon (varchar 15)
  • Alamat (text)
  • Password (optional)
  • Foto Profil (jpg, png, gif, jpeg)

Fitur: Real-time photo preview, validasi file, response JSON

🗄️ Perubahan Database

✅ Kolom baru ditambahkan ke tabel users:
Kolom Tipe Keterangan
email VARCHAR(100) Email address pengguna
no_hp VARCHAR(15) Nomor telepon pengguna
alamat TEXT Alamat lengkap pengguna
foto_profil VARCHAR(255) Nama file foto profil
⚠️ Setup Database: Jalankan setup_profile.php untuk menambahkan kolom-kolom ini

🔧 Troubleshooting

Database Error saat Setup

Solusi: Jalankan ulang setup_profile.php atau periksa koneksi di config.php

Upload Foto Tidak Berfungsi

Solusi: Pastikan folder 'uploads' writable (chmod 755)

Modal Tidak Muncul

Solusi: Buka browser console (F12) untuk melihat error JavaScript

Produk Tidak Muncul di Home Page

Solusi: Pastikan data produk ada di database products table

Dashboard Tidak Bisa Diakses

Solusi: Login ulang atau clear session browser
